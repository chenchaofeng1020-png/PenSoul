**1. 定义模型常量**
在 `SettingsPage.jsx` 中定义系统内置的模型选项列表：
- 豆包 1.5 Pro (`doubao-pro-1.5`)
- DeepSeek Chat (`deepseek-chat`)

**2. 改造 Agent 编辑界面**
在 `AgentManager` 组件的编辑区域（System Prompt 下方）增加“模型设置”模块：
- **模型选择**：添加下拉菜单，支持选择内置模型或“自定义模型”。
- **自定义输入**：当选择“自定义模型”时，显示文本输入框允许用户手动输入模型 ID。
- **参数同步**：选择模型后，自动更新 `model_config.model` 字段。

**3. 优化 JSON 配置区域**
- 将原有的“模型配置 (JSON)”区域保留，作为“高级参数配置”。
- 可以在此处配置 `temperature`、`max_tokens` 等其他高级参数。

**4. 优化详情展示**
- 在非编辑模式下的详情页，直观展示当前配置的模型名称（如显示“豆包 1.5 Pro”而不是 ID）。
